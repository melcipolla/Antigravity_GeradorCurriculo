# Gerador de Currículo ⚡ (Neon Edition)

Bem-vindo ao **Gerador de Currículo PRO**, uma aplicação web de alta performance desenvolvida em React para criar currículos magnéticos e profissionais em minutos, com um design **Dark Neon** de tirar o fôlego.

![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)

---

## ✨ Funcionalidades Principais

- **🎨 Design Premium Dark Neon**: Interface moderna com contraste em laranja neon e cinza escuro, otimizada para uma experiência visual impactante.
- **📄 Exportação PDF de Alta Fidelidade**: Gere arquivos PDF em formato A4 perfeitamente formatados usando `jsPDF` e `html2canvas`.
- **⚡ Fluxo em 5 Etapas**: Guia passo a passo (Dados Pessoais, Objetivo, Formação, Experiência e Habilidades) para garantir que você não esqueça nada importante.
- **🚀 Visualização em Tempo Real**: Alterne entre templates (Moderno e Clássico) e veja as mudanças instantaneamente antes de baixar.
- **🔐 Privacidade Total**: Seus dados são salvos apenas localmente no seu navegador (`localStorage`).

---

## 🛠️ Tecnologias Utilizadas

- **React 18** (Vite)
- **Tailwind CSS** (Estilização baseada em tokens)
- **Lucide React** (Ícones modernos)
- **jsPDF + html2canvas** (Geração de documentos)

## 🚀 Como Executar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone <url-do-repositorio>
   cd resume-generator
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador:**
   [http://localhost:5173](http://localhost:5173)
